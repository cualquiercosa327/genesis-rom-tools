Aridia Readme

Thanks for downloading Aridia! Here's what's included in this package:
  • aridia.exe - The main executable, just double-click.
  • MegaDriveIO.dll - Required DLL, must be in the same folder as aridia.exe (unless you're really clever and copy it to the GAC).
  • TileEditor.dll, PaletteEditor.dll - Same deal.
  • \data\ - Folder containing address locations of what can be edited in the rom, must be under the folder containing aridia.exe.
Please read the tutorial provided on the Aridia homepage before sending any questions. It explains how to use all the functions in the program and even discusses how to edit the source code 'n stuff.

Requirements
  • .NET Framework 1.1 (if you have XP or Vista then you probably already have this).
  • Phantasy Star III rom image (binary format), "Phantasy Star 3 - Generations of Doom (UE) [!].bin" in GoodGen, make a backup copy before editing, do not bother asking me for a copy.
  • Visual Studio .NET 2003+ (or Visual Studio Express) if you want to edit or build the source code.
