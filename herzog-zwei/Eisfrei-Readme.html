Eisfrei 1.1 Readme

Thanks for downloading Eisfrei! Here's what's included in this package:
  • eisfrei.exe - The main executable, just double-click.
  • MegaDriveIO.dll - Required DLL, must be in the same folder as eisfrei.exe (unless you're really clever and copy it to the GAC).
  • TileEditor.dll - Same deal as MegaDriveIO.dll.
  • \data\ - Folder containing address locations of what can be edited in the rom, must be under the folder containing eisfrei.exe.
Please read the tutorial provided on the Eisfrei homepage before sending any questions. It explains how to use all the functions in the program and stuff.

Although I test each release it's still entirely possible for Eisfrei to have a couple bugs. Not "it'll crash your computer" bugs but "something weird happened" bugs. Please feel free to report them. If you're planning a large hacking project I'd recommend taking a few backups along the way just to be safe.

The graphics and sprite editing could be a lot easier too, that will likely be something I address in a future release.

Requirements
  • .NET Framework 1.1 (if you have XP or Vista then you probably already have this).
  • Herzog Zwei rom image (binary format), "Herzog Zwei (UE) [!].bin" as recognized by GoodGen, make a backup copy before editing, do not bother asking me for a copy.
  • Visual Studio .NET 2003+ (or Visual Studio Express) if you want to edit or build the source code.
